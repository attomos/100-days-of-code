Disk Usage Before Yarn Workspaces

In my #100DaysOfCode projects, I included Tailwind CSS in many days (day 4, 5, 6, 7).
This introduces node_module folders on each one of these folders with pretty much duplicate content.

Disk Usage After Yarn Workspaces

As you can see from the screenshots below, disk usage went down drastically.
It puts all dependencies as zip files under .yarn/cache folder.
As a result, my packages (day 4, 5, 6, 7, and 13) are now contain only source code.
